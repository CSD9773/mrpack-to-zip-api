Modrinth Pack to Zip Converter API

API Usage

This API converts Modrinth modpacks (.mrpack) to standard ZIP format.

Use one of the following URL parameters:

  • ?project=<project_id> - Convert the latest version of a Modrinth project
  • ?url=<mrpack_url> - Convert a specific .mrpack file from a URL
  • &raw=true - Add this parameter to enable direct downloads with curl or wget

Example: https://domain.com/abcd/?project=1KVo5zza

For curl: curl -L "https://domain.com/abcd/?project=1KVo5zza&raw=true" -o modpack.zip

Note: Due to CORS, some linked mods that are hosted outside Modrinth may not properly download.
